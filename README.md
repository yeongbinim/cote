## 커밋 컨벤션

```
플랫폼(난이도): 문제제목(핵심 키워드)  
```

- `PRG`: 프로그래머스(programmers)
- `BOJ`: 백준(baekjoon online judge)
- `LCD`: 릿코드(Leet Code)
- `GRM`: 구름(goorm devth)
- `ETC`: 기타  

> ex) 프로그래머스 Level 2 디펜스 게임, Priority Queue 사용
> ```
> PRG(L2): 디펜스게임(PriorityQ)
> ```
